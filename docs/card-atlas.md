>

## 使用指南

- **卡牌奖励时查阅**：遇到 `card_reward` 状态时，读取本文件搜索候选卡牌。
- **评级标准**：S (必拿) / A (强力) / B (有用) / C (视情况) / D (一般跳过) / F (避免)
- **升级提示**：标注是否值得升级，以及升级后的关键变化。
- **按角色分区**：每个角色独立的卡池章节。通用牌（诅咒/状态/无色）单独分区。

## 评级调整规则

- **新卡初始评级**：基于首次遭遇时的效果估算，标注 `[初评]`
- **升级评级条件**：3+ 局中该卡表现优于当前评级 → 上调
- **降级评级条件**：2+ 局中该卡表现低于当前评级 → 下调
- **原型相关**：同一卡在不同原型中评级可能不同，用"最佳原型"列标注
