# STS2 — 敌人图鉴 (Bestiary)

> **温记忆层**。遇到敌人时按需读取，不自动加载。
>
> **更新时机**：每次遭遇新敌人或发现新应对策略后，在赛后反思中更新。

---

## 使用指南

- **战斗前查阅**：进入战斗时，读取本文件搜索敌人名称。
- **角色应对**：同一敌人对不同角色有不同最优解，每个角色单独列出。
- **危险评级**：★ (低) → ★★★★★ (极高)，基于平均 HP 损失。
- **遭遇记录**：`[W]` = 胜利, `[L]` = 失败, 标注 Run 编号。

## Boss

> **战术详情见 `sts2-combat-strategy.md` Act 1 Boss 章节**（热记忆，战斗时自动加载）。
> 本节仅记录 HP、核心机制摘要和遭遇历史。

### 史莱姆 Boss (Slime Boss)
- **HP**: 140 | **Act**: 1 Boss | **危险评级**: ★★★★
- **核心机制**: HP 降到 50% → 分裂为 2 只大史莱姆。塞黏液状态牌。
- **遭遇记录**: [Wiki] — 核心: 速打分裂+消耗黏液

### 守护者 (The Guardian)
- **HP**: ~240 | **Act**: 1 Boss | **危险评级**: ★★★★
- **核心机制**: 攻击模式 ↔ 防御模式（尖锐外壳反伤）切换。
- **遭遇记录**: [Wiki] — 核心: 防御模式不攻击+setup后爆发

### 六角幽灵 (Hexaghost)
- **HP**: 173/183 | **Act**: 1 Boss | **危险评级**: ★★★★★
- **核心机制**: 首回合 Divider 伤害按出牌数定。后续炼狱堆灼烧。
- **遭遇记录**: [Wiki] — 核心: 首回合少出牌+速杀

### 蠕虫 Boss (Worm Boss)
- **HP**: ~160 | **Act**: 1 Boss | **危险评级**: ★★★★★
- **核心机制**: 每回合塞大量状态牌，卡死正常抽牌和无限。
- **遭遇记录**: [Wiki] — 核心: 消耗清状态+速开无限

### 女王 (Queen)
- **HP**: 400/419 | **Act**: 1 Boss | **危险评级**: ★★★★★
- **核心机制**: 每回合锁定手里前几张牌，干扰连段。
- **遭遇记录**: [Wiki] — 核心: 绕过锁定+速开无限

### 墨影幻灵 (Vantom)
- **HP**: 173 | **Act**: 1 Boss | **危险评级**: ★★★★★
- **核心机制**: 9 层滑溜 + 伤口填充 + 力量成长 (+2/增益回合)
- **通用策略**: 多段攻击剥滑溜 → 消耗清伤口 → 速战速决
- **角色应对**:
  - **铁甲战士**: 双重打击/旋风斩剥滑溜，真实勇气/燃烧契约消耗伤口。薪火之源 T1 打出。
- **遭遇记录**: [L] Run #1 — 伤口填充 + 缺消耗机制 + 滑溜剥离太慢致死

## 精英 (Elite)

### 旧日雕像 (Bygone Effigy)
- **HP**: 127 | **Act**: 1 Elite
- **核心机制**: 休眠 → 增益 (+10 力量，分 2 回合) → 25+ 伤害攻击。拥有 Slow (每打一张牌 +10% 攻击伤害)。
- **危险评级**: ★★★★
- **通用策略**: 休眠/增益回合同时 setup 能力牌 + 输出。非攻击牌先打（叠 Slow 计数），攻击牌最后打享受 Slow 加成。
- **角色应对**:
  - **铁甲战士**: 势不可当+ 需要搭配能量成长才有效。优先力量叠加 + 多段攻击速杀。
- **遭遇记录**: [L] Run #2 — HP 不足 (63/80) 进入，25 伤害连击无法承受

## 普通怪 (Monster)

### 史莱姆群 (Slimes)
- **HP**: 多目标 | **Act**: 1
- **核心机制**: 大量黏液状态牌填充抽牌堆，消耗能量和手牌位。叶史莱姆添加状态，枝史莱姆造成伤害。
- **危险评级**: ★★★
- **通用策略**: 优先击杀叶史莱姆 (添加状态牌的)，枝史莱姆 (伤害型) 后杀。AOE 效率高。
- **角色应对**:
  - **铁甲战士**: 旋风斩 / 顺劈斩处理群体。有消耗源 (真实勇气) 可清理黏液。
- **遭遇记录**: [W-损耗] Run #2 — 5+ 黏液填充导致大量 HP 损失

### 毛绒伏地虫 (Fuzzy Wurm Crawler)
- **HP**: 57 | **Act**: 1 Monster
- **核心机制**: 交替攻击/增益。增益回合叠大量力量(7+)，后续攻击伤害急剧上升。
- **危险评级**: ★★★
- **通用策略**: 增益回合是免费输出窗口，必须全力进攻。不要让战斗拖超过3-4回合，否则力量成长致命。易伤提高击杀速度。
- **角色应对**:
  - **铁甲战士**: 放血+高费牌在增益回合最大化输出。痛击施加易伤后集中火力。
- **遭遇记录**: [W-大损耗] Run #3 — 战斗5回合，敌人叠7力量，结束时仅剩16 HP

### 墨宝 (Inklet)
- **HP**: 11-17 | **Act**: 1 Monster (群体出现×3)
- **核心机制**: 每只有滑溜(1)——首次受伤只扣1HP。3只同时攻击(每只3-6伤害)，总伤害12-15/回合。
- **危险评级**: ★★★ (群体压力大)
- **通用策略**: 多段攻击(双重打击等)一张牌剥滑溜并造成真实伤害。AoE对滑溜效率极低(每只只扣1HP)。集火最弱目标先杀减总伤害。
- **角色应对**:
  - **铁甲战士**: 双重打击是核心解答——剥滑溜+伤害。焚烧AoE清滑溜后下回合用攻击收割。优先杀低HP墨宝。
- **遭遇记录**: [L] Run #3 — 入战仅16HP，3只滑溜+高总伤害致死
