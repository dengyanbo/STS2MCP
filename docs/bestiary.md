# STS2 — 敌人图鉴 (Bestiary)

> **温记忆层**。遇到敌人时按需读取，不自动加载。
>
> **更新时机**：每次遭遇新敌人或发现新应对策略后，在赛后反思中更新。

---

## 使用指南

- **战斗前查阅**：进入战斗时，读取本文件搜索敌人名称。
- **角色应对**：同一敌人对不同角色有不同最优解，每个角色单独列出。
- **危险评级**：★ (低) → ★★★★★ (极高)，基于平均 HP 损失。
- **遭遇记录**：`[W]` = 胜利, `[L]` = 失败, 标注 Run 编号。

## Boss

### 墨影幻灵 (Vantom)
- **HP**: 173 | **Act**: 1 Boss
- **核心机制**: 9 层滑溜 (每次攻击只造成 1 伤害直到剥离) + 伤口填充 + 力量成长 (+2/增益回合)
- **危险评级**: ★★★★★
- **通用策略**: 多段攻击剥滑溜 → 消耗机制清伤口 → 速战速决（每拖一回合敌人 +2 力量）
- **角色应对**:
  - **铁甲战士**: 双重打击/旋风斩剥滑溜，真实勇气/燃烧契约消耗伤口。薪火之源 T1 打出。
- **遭遇记录**: [L] Run #1 — 伤口填充 + 缺消耗机制 + 滑溜剥离太慢致死

## 精英 (Elite)

### 旧日雕像 (Bygone Effigy)
- **HP**: 127 | **Act**: 1 Elite
- **核心机制**: 休眠 → 增益 (+10 力量，分 2 回合) → 25+ 伤害攻击。拥有 Slow (每打一张牌 +10% 攻击伤害)。
- **危险评级**: ★★★★
- **通用策略**: 休眠/增益回合同时 setup 能力牌 + 输出。非攻击牌先打（叠 Slow 计数），攻击牌最后打享受 Slow 加成。
- **角色应对**:
  - **铁甲战士**: 势不可当+ 需要搭配能量成长才有效。优先力量叠加 + 多段攻击速杀。
- **遭遇记录**: [L] Run #2 — HP 不足 (63/80) 进入，25 伤害连击无法承受

## 普通怪 (Monster)

### 史莱姆群 (Slimes)
- **HP**: 多目标 | **Act**: 1
- **核心机制**: 大量黏液状态牌填充抽牌堆，消耗能量和手牌位。叶史莱姆添加状态，枝史莱姆造成伤害。
- **危险评级**: ★★★
- **通用策略**: 优先击杀叶史莱姆 (添加状态牌的)，枝史莱姆 (伤害型) 后杀。AOE 效率高。
- **角色应对**:
  - **铁甲战士**: 旋风斩 / 顺劈斩处理群体。有消耗源 (真实勇气) 可清理黏液。
- **遭遇记录**: [W-损耗] Run #2 — 5+ 黏液填充导致大量 HP 损失
