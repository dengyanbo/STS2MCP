## 使用指南

- **遗物选择时查阅**：遇到 `relic_select`、商店遗物、事件遗物时读取本文件。
- **评级标准**：S (游戏改变级) / A (强力) / B (有用) / C (视情况) / D (一般不取)
- **标注规则**：`[通用]` = 所有角色受益，`[角色名]` = 特定角色最佳。
- **同一遗物不同角色可有不同评级**。

## Boss 遗物选择指南

> Boss 遗物通常有 3 选 1，以下为通用优先级原则：
>
> 1. **能量遗物** (每回合 +1 能量) — 几乎永远是最佳选择
> 2. **抽牌遗物** (每回合 +1 抽牌) — 仅次于能量
> 3. **条件型强力遗物** — 视牌组原型判断
> 4. **有负面效果的遗物** — 仅当正面效果足以补偿时

*(Boss 遗物具体评价将在首次遭遇后补充)*
